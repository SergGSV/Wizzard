Використання:

Створіть файл:

bashnano ssh-login.sh

Скопіюйте скрипт та відредагуйте масив SERVERS під ваші сервери
Надайте права на виконання:

bashchmod +x ssh-login.sh

Запустіть:

bash./ssh-login.sh
Особливості:

✅ Кольоровий інтерфейс
✅ Валідація введених даних
✅ Підтримка нестандартних портів
✅ Можливість виходу (q)
✅ Інформативні повідомлення
✅ Перевірка статусу підключення

Для додавання нового сервера просто додайте рядок у масив SERVERS:
bash"Назва сервера|користувач|адреса|порт"
